# Code of Conduct

Codex Designer is intended to be a welcoming project for everyone.

## Our standards

- Be respectful and constructive in all discussions and code reviews.
- Focus on behavior and ideas, not people.
- Be clear about assumptions, limitations, and tradeoffs.
- Keep collaboration practical and evidence-based.

## Unacceptable behavior

- Harassment, personal attacks, or discriminatory language.
- Sharing or requesting private information.
- Deliberately disruptive or unsafe tooling behavior.
- Misusing privileged features in ways that can damage workspaces/data.

## Scope

These standards apply in issues, PR discussions, and any collaborative environments associated with this repository.

## Enforcement

Project maintainers may remove, edit, or ban inappropriate contributions and interactions.
